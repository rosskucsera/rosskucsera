Website Pattern Library & 2014 Redesign

Overview

PeopleMatter's implementation and training team conducts training and setup with new customers. This can range from having one-on-one calls, delivering group webinars, creating how-to guides and more depending on the customer's size and needs.

Problem

Training new customers requires a significant time commitment from implementation team members. Some of these can be automated or made into on-demand resources if there was a place to house and explain the content.

Solution

Worked with implementation team members to develop and build a single place for new customers to access on-demand and personalized training, reducing burden on internal implementation team.

Skills used

  • Planning
  • Wireframing/Design
  • HTML/CSS
  • Support
